# 启动前端项目 npm start
# 启动模拟数据 npm run mock

## 没有用脚手架,用webpack从0搭建
## 样式:less(没有使用组件模块化)
## 状态管理:redux + react-redux + redux-thunk + immutable
## 路由:react-router
## 数据交互工具:fetch
## 数据模拟:koa搭建的服务器读取静态数据
